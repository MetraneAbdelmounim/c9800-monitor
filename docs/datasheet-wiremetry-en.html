WireMetry

Wi-Fi network monitoring & analytics platform — multi-vendor

WireMetry is an on-premise monitoring, troubleshooting and decision-support solution for controller-based (WLC) Wi-Fi infrastructures. It aggregates the live status of access points, clients and the RF environment in real time.

Designed to be vendor-agnostic, it supervises Cisco Catalyst 9800 and Ruckus SmartZone controllers from a single web interface, detects security and performance anomalies, and delivers concrete optimization recommendations.

Key points

  • Cisco Catalyst 9800 & Ruckus SmartZone, single interface
  • RF diagnostics: channel conflicts, spectrum, airtime, noise
  • Security log: rogues & anomalies, acknowledgeable
  • Network advisor: prioritized recommendations
  • Docker deployment, HTTPS — your data stays on-site
Real-time monitoring
  • Dashboard: CPU, memory, interfaces, controller version, AP & client counts, WLAN status.
  • Access-point inventory: model, status, version, channel, band, RF load, clients.
  • Clients: RSSI, SNR, channel, radio type; search and statistics.
  • Controller connectivity: connected / degraded / unreachable.
Site mapping

Import floor plans and place access points, with overlays:

  • Live AP status
  • Signal heatmap
  • Roaming paths
  • Coverage / channels
  • Client locator
RF diagnostics
  • Channel conflicts: co-channel, adjacent / overlapping interference.
  • Airtime utilization and noise level per radio.
  • Spectrum analyzer per band (2.4 / 5 GHz) with channel width.
Security event log
  • Detection: rogue APs, RF events, SNR drops, duplicate addresses, aWIPS alerts, offline APs.
  • Persistence in database, acknowledgement and history.
Decision support

Trends & capacity

Load, clients and RF occupancy over time — saturation forecasting.

Lifecycle & firmware

AP reboot / flap tracking and firmware compliance vs a target version.

Network advisor

Prioritized recommendations: RF conflicts, saturation, offline APs, firmware, weak clients, 2.4 GHz imbalance, controller load.

Tracking & roaming

Client connection and roaming history, with scheduled purge (5 min / hourly / daily / weekly / monthly) to keep data volume under control.

Security
  • Secure-token authentication, session expiry, admin / viewer roles.
  • Strong password hashing, forced rotation on first login.
  • Brute-force protection (login attempt limiting).
  • Encryption at rest of controller credentials.
  • Authenticated database, not publicly exposed.
  • Hardened web security headers (anti-clickjacking, HSTS).
  • End-to-end TLS encryption; runs as a non-privileged container.
